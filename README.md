# Oxford Burger Battle

GitHub Pages-ready mobile burger scorecard.

## Included files
- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`
- `Code.gs` for optional future Google Sheets sync

## Current UI
- Single-column scorecard
- Reviewer dropdown
- Restaurant order updated
- Ole Miss-inspired light blue and red palette
- Arial font
- Two action buttons only: Save and My Reviews

## Deploy on GitHub Pages
1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Set Source to **Deploy from a branch**.
5. Choose **main** and **/(root)**.
6. Save.

## Notes
- Reviews save locally in the browser on that device.
- `Code.gs` is included only for future central sync work.
